# 웨네버 운영 정책 초안

## 서비스 이용 조건

- 웨네버는 학교와 학년/반 프로필을 등록하고 학생증 인증이 승인된 사용자만 사용할 수 있습니다.
- 운영자는 허위 인증, 타인 사칭, 개인정보 노출, 괴롭힘, 광고/도배, 불법 또는 학교 커뮤니티 안전을 해치는 활동을 제한할 수 있습니다.
- 신고가 누적된 게시글과 댓글은 자동 숨김 처리될 수 있고, 관리자는 숨김, 복구, 삭제, 사용자 정지를 수행할 수 있습니다.

## 개인정보 처리

- 필수 수집 항목: 이메일, 이름, 학교, 학년, 반, 학생증 이미지, 시간표 이미지 또는 OCR 결과, 시험별 성적 점수 제보, 게시글/댓글/신고 기록, 알림 토큰, 광고 노출/클릭/숨김/신고 기록.
- 이용 목적: 회원 인증, 학교 커뮤니티 제공, 시간표/급식/댓글 알림, 익명 성적 제보 현황과 AI 참고 예측 제공, 신고 처리, 부정 사용 방지, 비개인화 광고와 시험기간 스폰서 캠페인 운영.
- 알림 토큰은 사용자가 알림을 켠 경우에만 저장하며, 알림을 모두 끄거나 탈퇴하면 삭제 대상입니다.

## 광고와 스폰서십

- 광고에는 명확한 `AD` 또는 `광고` 라벨을 표시합니다.
- 광고 타겟팅은 학교, 지역, 시험기간 같은 단위로만 운영하며 개인 성적, 반, 시간표, 학생증 이미지는 광고 타겟팅에 사용하지 않습니다.
- 부적절한 광고는 사용자가 숨기거나 신고할 수 있고, 신고 기록은 광고 품질 관리와 차단 처리에 사용합니다.
- 앱 실행 광고와 결과 진입 광고는 닫기 수단을 명확히 제공하고, 학생증 인증 전에는 노출하지 않습니다.

## 성적 점수 제보

- 성적 점수는 같은 학교/학년/과목/시험명 단위의 익명 집계와 참고 예측 목적으로만 사용합니다.
- 다른 사용자에게는 이름, 이메일, 사용자 ID 없이 익명 점수 목록과 집계값만 표시합니다.
- 제보 수가 5명 미만이면 개별 점수 목록과 상위 점수 통계를 표시하지 않습니다.
- AI 참고 예측에는 학교명, 이름, 이메일, 사용자 ID를 보내지 않고 과목명, 시험명, 만점, 전체 응시자 수, 익명 점수 목록만 사용합니다.

## 학생증 이미지 보관

- 학생증 이미지는 학생 인증 검수와 이의제기 대응 목적으로만 사용합니다.
- 앱에서는 signed URL로 제한된 시간 동안만 표시합니다.
- 승인 또는 반려 후 운영 보관 기간이 끝난 이미지는 삭제 대상이며, 탈퇴 시 즉시 삭제 대상입니다.
- 학생증 이미지는 공개 게시판, 프로필, 친구 기능에 노출하지 않습니다.

## 탈퇴와 데이터 삭제

- 탈퇴 요청은 `delete-account` Edge Function에서 처리합니다.
- 삭제 대상: Supabase Auth 사용자, 프로필, 인증 요청, 게시글/댓글/신고/좋아요, 시간표, 성적 점수 제보, 알림 설정, 학생증 이미지, 시간표 업로드 이미지.
- 법령상 보관이 필요한 신고 처리 기록이 생기면 별도 보관 기간과 항목을 고지해야 합니다.

## 문의

- 개인정보 열람/정정/삭제 요청, 신고 이의제기, 운영 문의는 앱 설정의 고객센터 이메일로 접수합니다.
